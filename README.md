- [x] Follow User
- [x] Unfollow User
- [x] Edit My profile
- [x] See user profile
- [x] See my profile
- [x] See the Full Photo
- [x] Upload a photo
- [ ] Edit a photo (Delete)
- [ ] See the feed
- [ ] See rooms

- [ ] Send private Message
